# Unit Economics
Формулы:
- CAC = marketing spend / new activated users
- CPA = spend / users completed first recommendation
- ARPU = revenue / all active users
- ARPPU = revenue / paying users
- Gross Margin = (Revenue - COGS) / Revenue
MVP focus: activation, retention, conversion readiness.
Пример: CAC $4, premium conversion 3%, ARPPU $8/month => payback требует улучшения retention.
Нужные данные: source channel, session completion, check-in repeat, premium view -> intent.
