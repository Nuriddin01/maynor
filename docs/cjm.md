# CJM
1. Awareness: пользователь видит бота в Telegram.
2. Onboarding: /start, понятное меню, дисклеймер wellbeing.
3. First value: night/day/power nap рекомендация.
4. Session execution: пользователь проходит шаги и ставит будильник.
5. Wake check-in: фиксирует эффект.
6. Habit loop: возвращается в историю/статистику.
Pain points: длинные формы, непонятные обещания, сложные таймеры.
Improvements: меньше шагов, быстрые кнопки, персонализация на данных.
