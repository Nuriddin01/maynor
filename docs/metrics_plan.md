# Metrics Plan
North Star: weekly helpful completed sessions.
Activation: % users who completed first recommendation in 24h.
Retention: D1/D7/D30.
Engagement: sessions per WAU, wake check-in completion rate.
Conversion: premium screen open -> premium intent.
Operational: alarm completion rate, stop code success rate.
SMART goals (90 days): D7 retention >= 25%, check-in completion >= 40%, alarm completion >= 75%.
Hypotheses: power nap UX improves daytime repeat usage; personalized audio increases helpfulness score.
Testing plan: weekly cohort review + A/B copy tests in prompts.
