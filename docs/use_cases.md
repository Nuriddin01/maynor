# Use Cases
- UC1 Night Wind Down: пользователь запускает ночной сценарий и получает персональную последовательность шагов.
- UC2 Day Recovery: пользователь выбирает дневной формат и получает recovery/nap рекомендацию.
- UC3 Power Nap: отдельный режим 10/15/20 минут.
- UC4 Alarm Management: установка/срабатывание/остановка кодом /stop_alarm CODE.
- UC5 Wake Check-in: сохранение пост-сессионного feedback.
- UC6 History & Stats: просмотр истории и персональных выводов.
- UC7 Settings: timezone, language, audio, reminders, default nap.
- UC8 Premium Stub: просмотр premium преимуществ и feature flags.
