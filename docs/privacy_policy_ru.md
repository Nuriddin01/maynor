# Политика обработки персональных данных
Сервис: Sleep Support Bot (wellbeing).

Собираем: telegram_id, имя (если есть), настройки, данные сессий сна и чек-инов, события использования.
Цели: работа рекомендаций, история, статистика, будильник, улучшение UX.
Минимизация: не собираем лишние медицинские данные и не обещаем лечение.
Хранение: SQLite MVP, доступ ограничен владельцем инфраструктуры.
Удаление: пользователь может запросить удаление данных через поддержку проекта.
Коммуникации: взаимодействие происходит внутри Telegram.
Согласие: использование бота означает согласие с этой политикой.
Дисклеймер: бот не заменяет врача; при устойчивых проблемах со сном обратитесь к специалисту.
