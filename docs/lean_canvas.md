# Lean Canvas — Sleep Support Bot
- Problem: сложность быстрого засыпания, неструктурированные дневные паузы, отсутствие простого sleep check-in.
- Customer Segments: офисные специалисты, студенты, удаленные сотрудники, молодые родители.
- UVP: «Мягкая поддержка сна в Telegram за 1-2 минуты без медицинских обещаний».
- Solution: night/day/power nap сценарии, будильник с кодом, персональные рекомендации, история и статистика.
- Channels: Telegram search, контент-маркетинг, партнерства с wellbeing-блогами.
- Revenue: freemium + monthly/annual premium.
- Cost Structure: infra, поддержка бота, контент сценариев, аналитика/эксперименты.
- Key Metrics: activation, session completion, wake check-in completion, D7 retention, premium conversion.
- Unfair Advantage: низкий friction Telegram UX + прозрачная recommendation logic.
